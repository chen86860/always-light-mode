# Chrome Web Store Listing — Long Description

Store console → Store listing → Description. One block per language.

## English (en)

Always Light Mode forces every website into light mode — even sites that ship their own dark theme.

Most "light mode" extensions only override your system preference. Always Light Mode neutralizes dark styling at every layer:

• Removes dark CSS rules (prefers-color-scheme: dark), including inside @import, @layer, @supports and @container
• Intercepts JavaScript theme detection (matchMedia) before the page loads, so anti-flicker scripts see "light" too
• Rewrites dark theme markers such as class="dark", data-theme, data-color-mode (GitHub) and data-bs-theme (Bootstrap)
• Works with modern sites: Shadow DOM, constructed stylesheets (CSS-in-JS) and iframes

SIMPLE CONTROLS
Click the toolbar icon to open the panel:
• Master switch — turn the extension on or off
• Site switch — exclude the current site with one click, in case it looks better untouched
• Email feedback — report a problematic site directly from the panel

PRIVACY
No tracking, no analytics, no data collection. Your settings sync through your browser account. Everything runs on your device.

Lightweight: no frameworks, just a few kilobytes.
